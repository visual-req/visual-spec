# зӣ®еҪ•з»“жһ„

[English](../en-US/structure.md) | [дёӯж–Ү](../zh-CN/structure.md) | [ж—Ҙжң¬иӘһ](../ja-JP/structure.md)

жң¬ж–ҮжҸҸиҝ°зӣ®ж ҮйЎ№зӣ®дёӯдҪҝз”Ёзҡ„зӣ®еҪ•з»“жһ„пјҡ
- Skill з”ҹжҲҗзҡ„ `/specs/` дә§зү©ж ‘пјҲйңҖжұӮ/и®ҫи®ЎдәӨд»ҳзү©пјү

## зӣ®ж ҮйЎ№зӣ®еёғеұҖ

Skill дјҡжҢүе‘Ҫд»Өйҳ¶ж®өйҖҗжӯҘеңЁ `/specs/` дёӢз”ҹжҲҗдә§зү©пјҢе№¶з»ҙжҠӨе°‘йҮҸйЎ№зӣ®зә§й…ҚзҪ®дёҺиҫ“е…ҘеҪ’жЎЈзӣ®еҪ•гҖӮе…ёеһӢзӣ®еҪ•з»“жһ„еҰӮдёӢпјҡ

```text
<your-project>/
в”ңв”Җ docs/                                  # иҫ“е…ҘеҪ’жЎЈпјҲ/vspec:new дјҡеҲӣе»әпјү
в”Ӯ  в”ңв”Җ legacy/                             # йҒ—з•ҷзі»з»ҹжқҗж–ҷпјҲеҠҹиғҪгҖҒжқғйҷҗгҖҒдәӨдә’гҖҒAPI зӯүпјү
в”Ӯ  в”ңв”Җ current/                            # жң¬ж¬Ўиҝӯд»Јж–°еўһ/иЎҘе……жқҗж–ҷпјҲд№ҹеҢ…жӢ¬ /vspec:new еј•з”Ёзҡ„еҺҹе§Ӣжқҗж–ҷж–Үд»¶пјҡPRD/йңҖжұӮж–ҮжЎЈ/жөҒзЁӢиҜҙжҳҺ/еӯ—ж®өеҸЈеҫ„/жЁЎжқҝ/еҚҸи®®/жҲӘеӣҫ/ж ·дҫӢж•°жҚ®зӯүпјү
в”Ӯ  в”Ӯ  в””в”Җ file_list.md                     # upgrade е…ҘеҸЈжё…еҚ•пјҲ/vspec:upgrade иҜ»еҸ–/з”ҹжҲҗпјү
в”Ӯ  в”ңв”Җ dependencies/                       # еӨ–йғЁдҫқиө–зі»з»ҹж–ҮжЎЈеә“пјҲе·Іжңүдҫқиө–зі»з»ҹиҜҙжҳҺпјҢжҺЁиҚҗпјү
в”Ӯ  в”ңв”Җ refine/                             # иЎҘе……/жҫ„жё…иҫ“е…ҘпјҲ/vspec:refine иҜ»еҸ–пјү
в”Ӯ  в”Ӯ  в””в”Җ file_list.md                     # еҸҜйҖүпјҡжңүеәҸиҫ“е…Ҙжё…еҚ•
в”Ӯ  в”ңв”Җ templates/                          # жЁЎжқҝпјҲPRD/ж–ҮжЎҲ/UI/иЎЁж јжЁЎжқҝпјҢеҸҜйҖүпјү
в”Ӯ  в”ңв”Җ texts/                              # ж–ҮжЎҲпјҲжҸҗзӨәиҜӯгҖҒй”ҷиҜҜдҝЎжҒҜгҖҒеҚҸи®®гҖҒж¶ҲжҒҜжЁЎжқҝпјҢеҸҜйҖүпјү
в”Ӯ  в””в”Җ assets/                             # йқҷжҖҒиө„жәҗпјҲи®ҫи®ЎзЁҝгҖҒжҲӘеӣҫгҖҒеҺҹеһӢеӣҫгҖҒеӣҫзүҮзӯүпјҢеҸҜйҖүпјү
в”ңв”Җ scheme.yaml                            # еҺҹеһӢйҖүж ҲпјҲ/vspec:verify дҪҝз”ЁпјӣзјәеӨұж—¶дјҡз”ҹжҲҗй»ҳи®ӨеҖјпјү
в””в”Җ specs/
   в”ңв”Җ background/                         # иғҢжҷҜеҲҶжһҗдёҺе…ұдә«жқҗж–ҷ
   в”Ӯ  в”ңв”Җ original.md                      # еҺҹе§ӢйңҖжұӮпјҲжҲ–зӯүд»·иҫ“е…Ҙпјү
   в”Ӯ  в”ңв”Җ stakeholders.md                  # е№Ізі»дәә
   в”Ӯ  в”ңв”Җ roles.md                         # и§’иүІдёҺиҒҢиҙЈ

   в”Ӯ  в”ңв”Җ scenarios.md                     # еңәжҷҜжё…еҚ•
   в”Ӯ  в”ңв”Җ scenario_details/                # еңәжҷҜиҠӮзӮ№з»ҶеҢ–пјҡpre_post.md/constraints.md/variations.md/boundaries.md/symmetry.md
   в”Ӯ  в”ңв”Җ dependencies.md                  # еӨ–йғЁдҫқиө–
   в”Ӯ  в””в”Җ questions.md                     # ејҖж”ҫй—®йўҳпјҲverify еүҚеҝ…йЎ»и§ЈеҶіжҲ–жҳҫејҸи·іиҝҮпјү
   в”ңв”Җ flows/                              # жөҒзЁӢеӣҫпјҲPlantUMLпјү

   в”Ӯ  в””в”Җ *.md
   в”ңв”Җ details/                            # еҚ•еҠҹиғҪиҜҰз»Ҷи®ҫи®ЎпјҲ/vspec:detail иҫ“еҮәпјү

   в”Ӯ     в”ңв”Җ file_import/
   в”Ӯ     в”ңв”Җ file_export/
```

иҜҙжҳҺпјҡ
- `docs/`пјҡеҪ’жЎЈдёҡеҠЎиҫ“е…ҘдёҺдәӨд»ҳеҜ№йҪҗжқҗж–ҷгҖӮе»әи®®дҪҝз”Ё `legacy/current/dependencies/refine/templates/texts/assets` зӯүеӯҗзӣ®еҪ•пјҢе№¶еңЁж–Үд»¶еҗҚдёӯеҢ…еҗ«жқҘжәҗ/зүҲжң¬дҝЎжҒҜгҖӮ
- `scheme.yaml`пјҡйҖүжӢ© `/vspec:verify` дёҺ `/vspec:impl` дҪҝз”Ёзҡ„жҠҖжңҜж ҲдёҺйҖүйЎ№гҖӮиҜ»еҸ–йЎәеәҸдёәйЎ№зӣ®ж №зӣ®еҪ•зҡ„ `scheme.yaml` дјҳе…ҲпјҢе…¶ж¬Ў `/specs/scheme.yaml`гҖӮиӢҘйғҪдёҚеӯҳеңЁеҲҷз”ҹжҲҗй»ҳи®Ө `scheme.yaml`гҖӮ

дёӢдёҖжӯҘпјҡ

- йҳ…иҜ» `commands.md`пјҢдәҶи§Јеҗ„ `/vspec:*` е‘Ҫд»Өзҡ„иҫ“е…ҘгҖҒиҫ“еҮәдёҺдҪҝз”ЁеңәжҷҜ
