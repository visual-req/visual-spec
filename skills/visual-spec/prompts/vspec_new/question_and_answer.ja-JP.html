VSpec Q&A ヘルパー（HTML + Markdown へ書き戻し）

目的：original.md と questions.md の質問をフォーム化（できるだけ選択式）し、回答後に markdown に書き戻します。 保存：File System Access API が使える場合は直接上書き、使えない場合はダウンロードして手動で差し替えます。

1) ファイル選択

通常のパス：/specs/background/original.md
通常のパス：/specs/background/questions.md
ヒント：questions.md だけ回答したい場合は、そのファイルだけ読み込んで保存できます。

2) 解析サマリー

original.md：未読み込み questions.md：未読み込み
選択肢の解析：Options: / 选项： / 選択肢： の後ろを選択肢として分割します（; / ； 区切り、または A. / 1) のリストに対応）。
保存（questions.md）：回答が入力されると Answer/回答 に書き込み、回答者/回答日時 を補完し、状態/Status を回答済みに更新します。

original.md：要確認事項

questions.md：Q&A 一覧
