VSpec 问答助手（HTML 交互 + 回写 Markdown）

作用：把 original.md 与 questions.md 里的提问渲染成表单（尽量按选择题），并在填写后回写到对应的 markdown 文件。支持两种保存方式：若浏览器支持文件写入 API，则直接覆盖写回；否则导出下载文件，由你手动替换。

1) 选择要回答的文件

目标路径通常是：/specs/background/original.md
目标路径通常是：/specs/background/questions.md
提示：若你只想回答 questions.md，可以只选择 questions.md 文件并保存。original.md 的“待确认问题”通常偏方向性澄清；questions.md 偏可追踪 Q&A。
